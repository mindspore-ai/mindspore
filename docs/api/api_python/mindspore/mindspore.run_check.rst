mindspore.run_check
===================

.. py:method:: mindspore.run_check()

    提供了便捷的API用以查询MindSpore的安装是否成功。

    **样例:**

    >>> import mindspore
    >>> mindspore.run_check()
    MindSpore version: xxx
    The result of multiplication calculation is correct, MindSpore has been installed successfully!
