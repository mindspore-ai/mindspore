mindspore.ops.Erfc
==================

.. py:class:: mindspore.ops.Erfc

    逐元素计算 `x` 的互补误差函数。

    更多参考详见 :func:`mindspore.ops.erfc`。

    输入：
        - **x** (Tensor) - 互补误差函数的输入Tensor。支持数据类型：

          - Ascend： float16、float32。
          - GPU/CPU： float16、float32、float64。

    输出：
        Tensor，具有与 `x` 相同的数据类型和shape。
