mindspore.ops.is_floating_point
================================

.. py:function:: mindspore.ops.is_floating_point(input)

    判断 `input` 的dtype是否是浮点数据类型，包括mindspore.flot64，mindspore.float32，mindspore.float16。

    参数：
        - **input** (Tensor) - 输入Tensor。

    返回：
        Bool，如果 `input` 的dtype是浮点数据类型，则返回True，否则返回False。
